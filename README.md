#  EcoVision

### An AI-powered assistive application designed to help visually impaired individuals understand their surroundings through computer vision and voice-based feedback.

---

##  Overview

EcoVision is an accessibility-focused AI application that helps visually impaired users identify objects in their environment.

The application uses computer vision techniques to analyze images captured through a smartphone and provides real-time audio feedback, allowing users to better understand nearby surroundings independently.

---

##  Problem Statement

Visually impaired individuals often face challenges in recognizing objects and navigating unfamiliar indoor environments.

EcoVision aims to provide an affordable and accessible solution by combining artificial intelligence with voice assistance technology.

---

##  Features

###  Object Detection
- Detects objects present in captured images
- Uses YOLO-based computer vision technology
- Identifies multiple objects simultaneously

###  Voice Feedback
- Converts detection results into audio responses
- Provides information about detected objects
- Improves accessibility and user interaction

###  User-Friendly Interaction
- Designed for simple operation
- Supports users with minimal technical interaction
- Focused on real-world usability
